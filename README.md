# IMKAK08

Source code pengerjaan IMKA 2016 oleh kelompok 8

1) ConnectToWifi.ino
    -> Connect to Access Point
    
2) KeypadPassword.ino
    -> Generate new password and testing password using keypad 4x4

3) SendData.ino
    -> Send data to localhost
    
4) SendDataThingSpeak.ino
    -> Send data (temperature) to API ThingSpeak
    
5) Motor.ino
    -> Run Motor DC clockwise and anti-clockwise by turns

6) Send2Data.ino
    -> Send data (two parameters) to localhost

7) UltrasonicESP8266.ino
    -> Send data to localhost based on the value of ultrasonic sensor

8) Infrared.ino
    -> Turn on LED when motion detects by Infrared LED and Photodiode
